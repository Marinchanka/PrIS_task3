Минимизировать $f(x,y)=(x-2)^2+(y+3)^2$ при




$g_1(x,y)=3x+2y-4=0,\quad g_2(x,y)=x- y-1=0.$

## 1. Идентификация задачи

Дана задача условной минимизации:

**Целевая функция:**  
$$
\[
f(x,y) = (x-2)^2 + (y+3)^2
\]  
$$
**Ограничения-равенства:**  
$$
\[
g_1(x,y) = 3x + 2y - 4 = 0
\]  
\[
g_2(x,y) = x - y - 1 = 0
\]
$$

Это задача квадратичной минимизации с двумя линейными ограничениями-равенствами.  
Метод множителей Лагранжа здесь применим, но так как ограничения линейные и их два, фактически они задают единственную точку (пересечение двух прямых), и условный экстремум будет именно в этой точке (если она существует).

Комментарий:
Задача определена корректно: целевая функция и оба ограничения выписаны верно. Правильно указано, что это квадратичная задача с двумя линейными ограничениями.
---

## 2. Решение методом множителей Лагранжа

Функция Лагранжа:  
\[
\mathcal{L}(x,y,\lambda_1,\lambda_2) = (x-2)^2 + (y+3)^2 + \lambda_1(3x + 2y - 4) + \lambda_2(x - y - 1)
\]

Необходимые условия экстремума (стационарности по \(x, y\) и условия ограничений):

\[
\frac{\partial \mathcal{L}}{\partial x} = 2(x-2) + 3\lambda_1 + \lambda_2 = 0 \quad (1)
\]  
\[
\frac{\partial \mathcal{L}}{\partial y} = 2(y+3) + 2\lambda_1 - \lambda_2 = 0 \quad (2)
\]  
\[
3x + 2y - 4 = 0 \quad (3)
\]  
\[
x - y - 1 = 0 \quad (4)
\]

---

### Шаг 1: Решаем систему ограничений (3) и (4)

Из (4): \(x = y + 1\).  
Подставляем в (3):  
\[
3(y+1) + 2y - 4 = 0
\]  
\[
3y + 3 + 2y - 4 = 0
\]  
\[
5y - 1 = 0 \quad \Rightarrow \quad y = \frac{1}{5}
\]  
\[
x = \frac{1}{5} + 1 = \frac{6}{5}
\]

Точка возможного экстремума:  
\[
(x^*, y^*) = \left(\frac{6}{5}, \frac{1}{5}\right)
\]

---

### Шаг 2: Находим множители Лагранжа

Подставим \(x=\frac{6}{5}\), \(y=\frac{1}{5}\) в (1) и (2):

(1):  
\[
2\left(\frac{6}{5} - 2\right) + 3\lambda_1 + \lambda_2 = 0
\]  
\[
2\left(\frac{6}{5} - \frac{10}{5}\right) + 3\lambda_1 + \lambda_2 = 0
\]  
\[
2\left(-\frac{4}{5}\right) + 3\lambda_1 + \lambda_2 = 0
\]  
\[
-\frac{8}{5} + 3\lambda_1 + \lambda_2 = 0 \quad (1')
\]

(2):  
\[
2\left(\frac{1}{5} + 3\right) + 2\lambda_1 - \lambda_2 = 0
\]  
\[
2\left(\frac{1}{5} + \frac{15}{5}\right) + 2\lambda_1 - \lambda_2 = 0
\]  
\[
2 \cdot \frac{16}{5} + 2\lambda_1 - \lambda_2 = 0
\]  
\[
\frac{32}{5} + 2\lambda_1 - \lambda_2 = 0 \quad (2')
\]

Складываем (1') и (2'):  
\[
\left[-\frac{8}{5} + 3\lambda_1 + \lambda_2\right] + \left[\frac{32}{5} + 2\lambda_1 - \lambda_2\right] = 0
\]  
\[
\frac{24}{5} + 5\lambda_1 = 0
\]  
\[
5\lambda_1 = -\frac{24}{5} \quad \Rightarrow \quad \lambda_1 = -\frac{24}{25}
\]

Из (1'):  
\[
-\frac{8}{5} + 3\left(-\frac{24}{25}\right) + \lambda_2 = 0
\]  
\[
-\frac{8}{5} - \frac{72}{25} + \lambda_2 = 0
\]  
\[
-\frac{40}{25} - \frac{72}{25} + \lambda_2 = 0
\]  
\[
-\frac{112}{25} + \lambda_2 = 0 \quad \Rightarrow \quad \lambda_2 = \frac{112}{25}
\]

---

Комментарии:
- Правильно идентифицирована цель и ограничения.
- Корректно найдена точка пересечения ограничений: x = y + 1, y = 1/5, x = 6/5.
- Правильно выписаны уравнения стационарности и правильно подставлены x,y.
- Решение системы для λ1, λ2 выполнено верно (все арифметические преобразования корректны).
- Гессиан Лагранжиана вычислен верно.
- Значение функции в найденной точке вычислено верно.

## 3. Проверка характера точки с помощью критерия Сильвестра

Критерий Сильвестра в задачах условной оптимизации применяется к **матрице Гессе функции Лагранжа** при ограничениях-равенствах.

Матрица Гессе функции Лагранжа по \((x,y)\):  
\[
H_{\mathcal{L}} = \begin{pmatrix}
\frac{\partial^2 \mathcal{L}}{\partial x^2} & \frac{\partial^2 \mathcal{L}}{\partial x \partial y} \\
\frac{\partial^2 \mathcal{L}}{\partial y \partial x} & \frac{\partial^2 \mathcal{L}}{\partial y^2}
\end{pmatrix}
= \begin{pmatrix}
2 & 0 \\
0 & 2
\end{pmatrix}
\]
(она постоянна и не зависит от \(\lambda_i\)).

---

### Критерий для условного экстремума

Пусть \(L = \nabla^2_{xx} \mathcal{L}\), \(J = \begin{pmatrix} \nabla g_1^T \\ \nabla g_2^T \end{pmatrix} = \begin{pmatrix} 3 & 2 \\ 1 & -1 \end{pmatrix}\).

Рассмотрим матрицу **расширенную матрицу Гессе** для случая \(m=2\) ограничений, \(n=2\) переменных:  
\[
\overline{H} = \begin{pmatrix}
0 & 0 & 3 & 2 \\
0 & 0 & 1 & -1 \\
3 & 1 & 2 & 0 \\
2 & -1 & 0 & 2
\end{pmatrix}
\]
(блок нулей \(m\times m\), блок \(J\), блок \(J^T\), блок \(L\)).

Правило знаков для условной оптимизации (для минимума при ограничениях-равенствах):  
Нужно проверять знаки последних \(n-m\) главных миноров начиная с порядка \(2m+1\), т.е. здесь \(n-m=0\) — случай вырожденный, фактически единственная точка.

---

Но более простой подход:  
Так как \(H_{\mathcal{L}}\) положительно определена (\(2>0, 4>0\)), то в точке \((x^*,y^*)\) имеем **условный строгий минимум**.

---

**Проверка:**  
Ограничения задают единственную точку, поэтому она автоматически является и глобальным минимумом (и максимумом) на допустимом множестве.  
Вычислим \(f(x^*,y^*)\):  
\[
f\left(\frac{6}{5}, \frac{1}{5}\right) = \left(\frac{6}{5} - 2\right)^2 + \left(\frac{1}{5} + 3\right)^2
\]  
\[
= \left(-\frac{4}{5}\right)^2 + \left(\frac{16}{5}\right)^2
\]  
\[
= \frac{16}{25} + \frac{256}{25} = \frac{272}{25}
\]

---

Комментарии:
а) Формулировка про «единственную точку — и минимум, и максимум»: это формально верно (на допустимом множестве, состоящем из одной точки, эта точка одновременно и минимум, и максимум). Но обычно нас интересует характер точки как экстремума функции среди допустимых точек; тогда правильнее сказать: на допустимом множестве, которое состоит из единственной точки, эта точка тривиально является и минимумом, и максимумом; однако по стандартным условиям вторых производных она является строгим условным минимумом.
б) Гессиан по переменным вычислен верно и является положительно определённым.
в) В тексте приведена расширенная матрица Гессе и затем сделан вывод «n-m=0 — вырожденный случай», дальше аргументация расплывчатая. В этой задаче n=m=2, поэтому допустимое линейное подпространство приращений имеет размер 0 и проверка через расширенный Гессиан тривиальна. Проще и корректнее записать: Поскольку Гессиан по переменным положительно определён и n=m (размер допустимого подпространства равен нулю), найденная точка является строгим условным минимумом.


## 4. Ответ
